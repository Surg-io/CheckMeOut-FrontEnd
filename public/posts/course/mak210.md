# **3D Printing & CNC Milling Course**  

## **Course Description**  
This course provides an introduction to **3D printing** and **CNC milling**, focusing on design preparation, machine operation, and troubleshooting. Students will gain hands-on experience using slicing and CAM software to fabricate projects.  

By the end of the course, students will be certified to use both the 3D printer and CNC milling machine independently in the makerspace.  

---

## **Course Objectives**  
Upon successful completion of this course, students will be able to:  
- Understand the fundamental principles of **3D printing** and **CNC machining**.  
- Prepare digital files using slicing software and CAM tools.  
- Operate 3D printers and CNC milling machines safely and efficiently.  
- Select appropriate materials and machine settings for different projects.  
- Troubleshoot common issues related to printing and milling.

## **Course Structure & Schedule**  

### **Week 1: Introduction to 3D Printing & CNC Milling**  
- Overview of additive (3D printing) and subtractive (CNC milling) manufacturing.  
- Machine components, safety protocols, and workspace setup.  

### **Week 2: Digital File Preparation**  
- 3D slicing software (Cura, PrusaSlicer, etc.).  
- CAM software (Fusion 360, VCarve, etc.) for CNC toolpaths.  

### **Week 3: Machine Operation & Calibration**  
- Hands-on 3D printing and CNC milling exercises.  
- Setting power, speed, and material adjustments.  

### **Week 4: Troubleshooting & Project Work**  
- Diagnosing common print/machining issues.  
- Preparing personal projects for final evaluation.  

### **Week 5: Final Project & Certification**  
- Students fabricate a final project using either or both machines.  
- Certification assessment based on independent machine operation.
